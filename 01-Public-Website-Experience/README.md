# 01. Public Website Experience Pipeline

This pipeline covers the public website as the discovery and conversion layer of the Fluke Games platform.

## Categories

| Category | Summary | README |
|---|---|---|

## Pipeline Summary

The public website pipeline connects the first visitor interaction to structured backend capabilities. It shows how page routing, content modules, interactive components, service clients, API Gateway, Lambda services, DynamoDB, external services, and AI chat work together as one coherent public experience.
