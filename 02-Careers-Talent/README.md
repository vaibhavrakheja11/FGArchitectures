# 02. Careers & Talent Pipeline

This pipeline covers the talent acquisition system from public role discovery to applicant review and automated communication.

## Categories

| Category | Summary | README |
|---|---|---|
| Website | Public jobs, dynamic application builder, applicant payloads, review workflow, email and WhatsApp notifications | [Open Website Architecture](Website/README.md) |

## Pipeline Summary

The careers pipeline turns public interest into structured hiring data. It connects job discovery, role-specific forms, applicant records, admin review, and candidate communication into one operational workflow.
