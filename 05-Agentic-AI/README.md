# 05. Agentic AI Pipeline

This pipeline groups all AI architecture variants under one numbered category. Agentic AI appears across the website, internal backend operations, InterviewPrepApp, and UnrealMCP workflows.

## Categories

| Category | Summary | README |
|---|---|---|
| Internal | LangGraph, LlamaIndex RAG, worker Lambda, memory, MCP tools, approval guardrails | [Open Internal Architecture](Internal/README.md) |
| Website | Public assistant and website-facing AI plans | [Open Website Architecture](Website/README.md) |
| InterviewPrepApp | Interview coaching and adaptive practice AI plans | [Open InterviewPrepApp Architecture](InterviewPrepApp/README.md) |
| UnrealMCP | Unreal Engine and MCP-driven game/tooling automation plans | [Open UnrealMCP Architecture](UnrealMCP/README.md) |

## Pipeline Summary

The agentic AI pipeline covers routed AI agents, context construction, memory, LangGraph workflows, LlamaIndex RAG, MCP actions, approval guardrails, async worker execution, and result delivery. It is intended to support both public experiences and internal productivity workflows.
